## 产品定位
管理人脉圈

## 产品解决的痛点
随着朋友，客户越来越多，需要一款软件来统一管理人脉圈

## 产品潜在客户
1. 大众用户

## 需求描述
1. 管理朋友圈
2. 记录与朋友的交流
3. 能根据技能，年龄等，做综合统计

## 需求分析
1. 朋友管理
2. 交流记录管理
3. 综合查询，

## 技术架构
1. Java：Nutz框架
2. 缓存：EhCache，
3. 静态模板框架：Beetl
4. 后台UI框架：B-JUI
5. 数据库：MySql
6. 后台服务器：阿里云centos6
